Command Window

A block-based REPL — no picker, no guided runner, no preview. The user types shell commands themselves; each becomes a discrete block that never summarizes or auto-closes.

01

Friction Contrast & Zero-Agent

Stopping a live block is deliberately harder — hold + confirm, explaining exactly what it affects. Command windows also fully support zero-agent sessions on their own.

$ npm run dev ● running · up 12s

Stop npm run dev?

Kills the process in tmux window kc-dev. The block stays, marked stopped, until dismissed. The agent and other blocks are unaffected.

RULES §6 · destructive friction, scoped explicitly — this kills one process, not the session.
