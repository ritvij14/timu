What is timu?

Connect your phone to any SSH-accessible machine — a VPS, a local Mac, anything with sshd.

Drive coding agents like Codex, Claude Code, and OpenCode through a WhatsApp-style chat UI.

Sessions live in a tmux pane on your machine, so they survive app closes, network drops, and switching devices.

No hosted backend. No cloud sync. Your machine, your code, your sessions.

Built for mobile coding

Everything you need to run AI coding sessions from your phone. Nothing you don't.

💬

Chat-first UI

Not a terminal on your phone. A WhatsApp-style chat that drives real coding agents.

Readiness checks

Folder picker

Browse projects on your machine and start a session in the right directory.

Rust core + FFI

A Rust library handles SSH, tmux, and SFTP. The Expo app drives it over FFI.

How it works

Three steps from zero to a coding session running on your machine.

1

Connect a machine

Pair your phone with any SSH-accessible machine using a one-time QR code.

2

Pick a folder + agent

Choose a project directory and launch Codex, Claude Code, or OpenCode.

3

Chat. Session persists.

Talk to the agent through chat. Close the app, reopen it, and your session is still there.
